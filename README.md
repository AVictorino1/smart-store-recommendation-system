# Smart Store Recommendation System

Proyecto personal enfocado en automatización inteligente para tiendas pequeñas.

## Tecnologías actuales
- Python
- FastAPI
- SQLite
- scikit-learn
- Pandas
- React
- Cohere API

## Funcionalidades actuales
- API REST básica
- Sistema de recomendaciones usando filtros, one hot encoding, embeddings (Cohere API), cosine similarity
- Backend modular
- Dashboard administrativo para visualizar base de datos (React)

## En desarrollo
- Dashboard administrativo con React
- Integración con Meta API (Whatsapp, Instagram, Facebook) 
- Creación de bases de datos personalizadas a través de imágenes de productos
- Recomendaciones visuales mediante análisis de imágenes
- Clasificación automática de productos mediante análisis de imágenes
