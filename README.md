# Smart Store Recommendation System

Proyecto personal enfocado en automatización inteligente para tiendas pequeñas.

## Tecnologías actuales
- Python
- FastAPI
- SQLite
- scikit-learn

## Funcionalidades actuales
- API REST básica
- Sistema de recomendaciones usando cosine similarity
- Backend modular

## En desarrollo
- Mejora base de datos, añadiendo más categorías, variantes, y expresiones naturales con Claude API
- Aplicación de TF-IDF, tokens, embeddings con PML etc.
- Dashboard administrativo con React
- Integración con WhatsApp API
- Recomendaciones visuales mediante análisis de imágenes
- Clasificación automática de productos mediante análisis de imágenes
